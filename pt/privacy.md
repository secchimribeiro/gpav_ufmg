# Política de Privacidade

Esta Política de Privacidade descreve como as informações são coletadas, utilizadas e protegidas no website do **Grupo de Pesquisa em Doenças Arbovirais**, vinculado ao Departamento de Morfologia, Instituto de Ciências Biológicas, Universidade Federal de Minas Gerais (UFMG).

## 1. Princípios gerais

Este website tem caráter **institucional, acadêmico e informativo**. Nosso compromisso é respeitar a privacidade dos visitantes e tratar quaisquer dados pessoais de forma responsável, transparente e segura, em conformidade com a **Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018)**.

## 2. Coleta de dados pessoais

Podemos coletar dados pessoais **apenas quando você os fornece voluntariamente**, especificamente por meio do formulário de contato disponível neste website. Os dados coletados podem incluir:

- Nome
- Sobrenome
- Instituição
- Endereço de e-mail
- Conteúdo da mensagem enviada

Não coletamos dados pessoais sensíveis.

## 3. Finalidade do uso dos dados

Os dados fornecidos são utilizados exclusivamente para:

- Responder a mensagens enviadas por meio do formulário de contato
- Manter comunicação institucional relacionada às atividades do grupo de pesquisa

Os dados **não são utilizados para fins comerciais**, marketing, ou envio de comunicações não solicitadas.

## 4. Compartilhamento de dados

Os dados pessoais coletados **não são compartilhados com terceiros**, exceto quando exigido por obrigação legal ou regulatória.

## 5. Armazenamento e segurança

As mensagens enviadas por meio do formulário de contato são encaminhadas para um endereço de e-mail institucional ou pessoal do responsável pelo site. São adotadas medidas razoáveis para proteger as informações contra acesso não autorizado, perda ou uso indevido.

No entanto, nenhum sistema é completamente seguro, e não podemos garantir segurança absoluta.

## 6. Cookies e tecnologias de rastreamento

Este website **não utiliza cookies próprios** para identificação pessoal dos usuários.

Podemos utilizar ferramentas de análise de tráfego (como Google Analytics) para fins estatísticos e de melhoria do website. Essas ferramentas coletam dados de forma agregada e anonimizada, sempre que possível.

## 7. Links para sites externos

Este website pode conter links para sites externos (por exemplo, revistas científicas, bases de dados ou redes sociais). Não nos responsabilizamos pelas políticas de privacidade ou pelo conteúdo desses sites.

## 8. Direitos do titular dos dados

Você tem o direito de:

- Solicitar acesso aos seus dados pessoais
- Solicitar correção ou exclusão de dados
- Revogar o consentimento para o uso dos dados

Solicitações podem ser feitas por meio do formulário de contato ou diretamente pelo e-mail informado no site.

## 9. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos que os usuários revisem esta página regularmente para se manterem informados sobre eventuais alterações.

## 10. Contato

Em caso de dúvidas sobre esta Política de Privacidade ou sobre o tratamento de dados pessoais, entre em contato por meio do formulário disponível neste website.
